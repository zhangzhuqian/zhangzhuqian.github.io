下载全网视频，我只用这个

2017-09-28

You-Get是一个基于 Python 3 的下载工具。使用 You-Get 可以很轻松的下载到网络上的视频、图片及音乐。使用方法快捷有效到只需要一步，并且速度喜人。目前是我的下载视频的主力方式，推荐给大家。

安装方法

you-get 安装可以使用 Python3的 pip3方法，必须提前安装Python3

$ pip3 install you-get

也可以使用包安装程序 brew，brew 会自动帮你搜索安装程序的依赖包并进行安装，brew 安装方法可直接谷歌，一行程序就可以安装完成，建议使用该方法

$ brew install you-get

使用方法

比如想要下载白夜追凶第一集，只需要在网站上找到对应的剧集的url：

http://v.youku.com/v_show/id_XMjk4ODAyMzIyOA==.html?spm=a2h0j.8191423.vpofficiallistv5_wrap.5~5~5~5~A

在命令行程序中输入

$ you-get http://v.youku.com/v_show/id_XMjk4ODAyMzIyOA==.html?spm=a2h0j.8191423.vpofficiallistv5_wrap.5~5~5~5~A

看到以下信息，就开始下载了，注意，可能需要稍作等待

下载注意

  1. 下载时会全速下载
  2. you-get 下载会将剧集切分成多部分来下载，在各个部分下载完成之后会进行

支持的网站

国内大部分视频网站都支持，比如：优酷，腾讯视频，爱奇艺，芒果TV，哔哩哔哩，乐视等这些主流网站
